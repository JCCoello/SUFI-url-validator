## Repo overview

This is the SUFI URL Validator repo, also used as the team's QA hub.

- `src/`, `tests/` — URL validator. Playwright spec ([tests/assets.spec.ts](../../../../tests/assets.spec.ts)) + standalone CLI ([src/validator.ts](../../../../src/validator.ts)). Validates a CSV of URLs against HTTP 200 with redirects followed (404 is terminal).
- `reports/QA/` — human-written QA docs, tracked in git:
  - `test-plans/` — test plans and strategy notes (this folder lives here)
  - `bug-reports/` — defect logs
  - `manual-runs/` — exploratory / manual session logs
- `reports/*.{json,csv}` — auto-generated validator output, gitignored.

Automated tests go under `tests/` — Playwright's `testDir` picks up any `**/*.spec.ts`.
